
# Code Test (react-native-expo-typescript)

This project was react-native-expo with typescript

## Getting started

### `npm install --global expo-cli`

Expo CLI is required. You can install it globally by running npm install --global expo-cli. Consult the Expo documentation for more information.

### Expo Go app for iOS and Android
- [Android Play Store](https://play.google.com/store/apps/details?id=host.exp.exponent) - Android Lollipop (5) and greater.
- [iOS App Store](https://itunes.com/apps/exponent) - iOS 11 and greater.

### `npm install`

Clone or download the repository, navigate to the extracted folder with a terminal and run `npm install` command.

### `npm start`

When you run npm start (or expo start), Expo CLI starts Metro Bundler and also pops up Expo Dev Tools, a graphical interface for Expo CLI.

### Opening the app on your phone/tablet

- On your iPhone or iPad, open the default Apple "Camera" app and scan the QR code you see in the terminal or in Expo Dev Tools.
- On your Android device, press "Scan QR Code" on the "Projects" tab of the Expo Go app and scan the QR code you see in the terminal or in Expo Dev Tools.
